# צפייה בטלפון

1. הפעל שרת במחשב:
   ```
   cd peymiz-pitch
   npm run dev
   ```

2. מצא את כתובת ה-IP של המחשב (אותה רשת Wi-Fi כמו הטלפון):
   - Windows: `ipconfig` → IPv4 Address

3. בדפדפן בטלפון פתח:
   ```
   http://[IP-שלך]:3456
   ```
   לדוגמה: `http://192.168.1.10:3456`

4. במחשב: http://localhost:3456
